What?

  • Generate 3D conformational ensembles on compound libraries (tedious)
  • Compare & contrast CPU-based methods with GPU-based methods of generation
  • Test different library generation parameters, e.g. shape, coloring
  • Use publicly available data from PubChem

Why?

  • Exploit GPU parallelism
  • Apply methodology to drug design & virtual screening

Complexity of Tasks

  • Not much domain knowledge
  • Reading, understanding papers requires a lot of refreshing chemistry knowledge
  • Using a lot of high-level frameworks, isolated from difficult GPU
  • Handling large data sets

Products Delivered

  • Applications for HPC resources, focus research goals
  • Applications and academic licenses for OpenEye ROCS, ChemAxon
  • Reports on read papers
  • Gathered, sorted through initial data

Computing and Other Knowledge Gained

  • The tremendous paperwork behind research
  • Reinforce basic knowledge of chemistry concepts, conformers
  • Basics of chemical informatics, libraries and how to handle data
  • Learn about the advantages of GPU-based computing techniques
  • Data analysis: similarity clustering

Tools and Techniques Used and Learned

  • PubChem with various chemical interchange formats and metadata
  • OpeneEye (Fast)ROCS, OMEGA
  • Python, OpenBabel
  • Shared computing resources
