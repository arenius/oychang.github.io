Step 1: Software Familiarization & Subset Trials

Detailed Process

  1. Download ~400K MLSMR compounds in Gzipped SDF, 1 3D conformer per CID (mlsmr.sdf)
  2. Pick a random subset of ~10K compounds as a test set with some Python script (mlsmr-subset.sdf)
  3. Generate a fragment library with OMEGA's makefraglib, to speed up conformation results with default settings, (makefraglib -in mlsmr-subset.sdf -out mslmr-subset-fraglib.oeb)
  4. Create several sets of conformers, with our pre-calculated fragment library with varying numbers of conformers, e.g. (1, 10, 25, 50, 100)
  5. Automate runs of ROCS to create an N x N similarity matrix
  6. Score the similarity matrices
  7. Determine the ideal number of conformers by observing the similarity matrices

Commands

n_conformers = 1 # some number
# Generate conformers with omega
omega2 -in mlsmr-subset.sdf \
       -out mlsmr-n_conformers-subset.oeb.gz \
       -addfraglib mlsmr-subset-fraglib.oeb \
       -maxconfs n_conformers \
       -progress percent
# ROCS for shape similarity
rocs \
–query \
OPENEYE_DIR/data/vrocs/simple/molecule_idea.sq \
–dbase \
mlsmr-n_conformers-subset.oeb.gz
